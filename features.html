I'm actively improving PerfPerTCO to make it even more powerful and user-friendly. Here's a preview of the features I'm working on:

Planned Features

  • Custom Upload Configurations Gallery: Users will be able to upload and benchmark their own GPU specs and benchmark data ((SON/CSV configuration import)
  • Multi-node Scaling Selector: Simulate large-scale cluster deployments with different user-defined node counts.
  • Currency Selector: Support for USD, GBP, INR, and more.
  • Benchmark Database Expansion: More workloads beyond GROMACS and AMBER.
  • Preset Profiles: Academic Cluster, etc.
  • Detailed PDF Report Generator: Export project report with charts and explanations.
  • Collaboration Mode: Share calculation setups with teammates via unique shareable public URLs with embedded settings.
  • API Access: Programmatic access to benchmark and calculation engine for researchers/devs.

Optional Features?

  • Carbon Footprint Estimator
  • Multi-Year TCO Projection with Depreciation Curves
  • Hardware Configuration Assistant
