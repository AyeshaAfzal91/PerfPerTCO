I'm actively improving PerfPerTCO to make it even more powerful and user-friendly. Here's a preview of the features I'm working on:

Planned Features

  • Benchmark Database Expansion: More workloads; currently only GROMACS and AMBER.
  • Preset Profiles: Top 10 Clusters from Top500 of Green500 list, etc.
  • Collaboration Mode: Share calculation setups with teammates via unique shareable public URLs with embedded settings; currently simple blog summary share option
  • Programmatic Access: Programmatic access to benchmark and calculation engine for researchers/devs; currently its private GitHub repo
  • Detailed PDF Report Generator: Export project report with charts and explanations; currently its simple PDF Report Generator.
  • Currency Selector: supporting USD, GBP, INR, and more; currently only EUR
  • Hardware Configuration Assistant; currently only simple Smart Strategy Tip
