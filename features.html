Current Use-case: The tool currently evaluates how various parameters affect overall Performance Per TCO. Additionally, it examines scenarios where GPU Type 1, though slower but more power-efficient than Type 2, could leverage energy savings to deploy more units and potentially match or exceed Type 2‚Äôs performance within the same budget and lifespan.

Future Use-case: With the new additions, it will help determine how performance and energy consumption are impacted, for example, when running GROMACS/AMBER on an A100 at 100 MHz instead of 1.5 GHz. Users can then decide what's more important to them: maximizing performance or finding the sweet spot for maximum performance per watt.

Tool Focus: A tool for exploring the parameter space to analyze sensitivity, rather than for directly planning procurement, since procurement involves many dynamic factors that cannot be captured by a single tool.

Here's a preview of the upcoming features I'm working on to make PerfPerTCO even more powerful and user-friendly:
